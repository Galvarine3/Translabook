## Despliegue

Este sitio está diseñado para ser desplegado como un sitio estático en [Render](https://render.com/).

### Configuración en Render:

1. Conecta este repositorio a tu cuenta de GitHub
2. Crea un nuevo "Static Site"
3. Configura:
   - **Build Command**: `echo "No build required"`
   - **Publish Directory**: `.`
   - **Publish Path**: `privacy-policy.html`

El sitio estará disponible en la URL que Render te asigne.
